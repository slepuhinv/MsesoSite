Пробная версия плагина для слабовидящих пользователей.

== Description ==

**Comfortable Reading** - демо версия плагина, которая создает версию вашего сайта для слабовидящих посетителей. Плагин позволяет пользователю изменить цветовую схему сайта и размеры шрифтов.

**Обратите внимание, что данная бесплатная версия плагина не подходит для сайтов, требующих полного соответствия ГОСТ, так как не умеет отключать изображения.**

С этой задачей справляется **[премиум версиия плагина](http://wp-lessons.com/comfortable-reading-premium "премиум версиия плагина")**, в которой эта возможность предусмотрена.

== Installation ==

1. Разархивируйте и закачайте папку плагина в 'wp-content/plugins/'
2. Зайдите в административное меню WordPress
3. Активируйте плагин
4. Зайдите в 'Внешний вид' > Виджеты
5. Добавьте виджет 'Comfortable Reading' в сайдбар (желательно в тот, который отображается на всех страницах)

== Frequently Asked Questions ==
