Демо версия плагина для слабовидящих.

== Description ==

Важно! - Изменение кода плагина приводит к неработоспособности.

= Плагин bvi v0.5.2 включает в себя: =
* Включение/Отключение изображения (в место изображения появится alt надпись);
* Изменение размера шрифта (14px, 18px, 23px);
* Изменение фона страницы (Черным по белому, Белым по черному, Темно-синим по голубому);
* Добавлять кнопку в виджетах;
* Поддержка браузеров (Chrome, Firefox, Safari, Internet Explorer(5, 6, 7, 8, 9, 10, 11), Microsoft Edge, Opera и тд.);

= Плагин bvi PRO v0.8 включает в себя: =
* Включение/Отключение изображения (в место изображения появится alt надпись);
* Сохранение ширины и высоты изображения при отключении (в место изображения появится alt надпись);
* Изменение размера шрифта (14px, 18px, 23px);
* Изменение фона страницы  (Черным по белому, Белым по черному, Темно-синим по голубому, Коричневым по бежевому, Зеленым по темно-коричневому);
* Изменение шрифта (Arial, sans-serif; Times New roman", serif;);
* Изменение шаблона кнопки (Версия для слабовидящих);
* Изменение интервала между буквами (Кернинг) : (Стандартный, Средний, Большой);
* Панель фиксируется вверху при прокрутки страницы;
* Включение/Отключение перезагрузки страницы (переход на обычную версию сайта);
* Админ панель с настройками;
* Добавлять кнопку в виджетах;
* Добавлять шорткод;
* Поддержка браузеров (Chrome, Firefox, Safari, Internet Explorer(5, 6, 7, 8, 9, 10, 11), Microsoft Edge, Opera и тд.);

== Installation ==

= Button visually impaired Pro v0.8 =
1. Закачайте плагин в директорию `/wp-content/plugins/`.
2. Плагин активировать через меню WordPress > Плагины > bvi Pro v0.8 > Активировать.
3. Перейдите bvi Pro v0.8 и в настройках включите плагин.
4. Добавьте кнопку для слабовидящих в виджете или воспользуйтесь шорткодом <code>[bvi template=big_icon_black]</code>.

= Button visually impaired v0.5.2 =
1. Закачайте плагин в директорию `/wp-content/plugins/`.
2. Плагин активировать через меню WordPress > Плагины > Button visually impaired v0.5.2 > Активировать.
3. Добавьте кнопку для слабовидящих в виджете.
